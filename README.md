# Stock News Sentiment Analysis

This project analyzes the sentiment of stock-related news articles fetched from Yahoo Finance's RSS feeds. The sentiment is classified as Positive, Negative, or Neutral using a pre-trained NLP model.

## Prerequisites

Before running the script, ensure you have the following installed:

- Python 3.7+
- `transformers` library
- `feedparser` library

Install dependencies using pip:
```bash
pip install transformers feedparser
